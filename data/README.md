# Data

## Sources

| File pattern | Source | Content |
|---|---|---|
| `raw/tvl_<slug>.csv` | `https://api.llama.fi/protocol/<slug>` | Daily protocol TVL (USD): `aave-v3`, `spark`, `morpho` |
| `raw/token_flows_<slug>.csv` | same endpoint, `tokensInUsd` field | Token-level TVL decomposition (stablecoin migration, Fig. 2) |
| `raw/apy_<pool_prefix>.csv` | `https://yields.llama.fi/chart/<pool_id>` | Daily pool APY for the four matched stablecoin pools |
| `processed/mechanism_panel.csv` | derived | Merged TVL × APY-stress protocol-day panel (regenerated by pipeline) |

Pool IDs and protocol slugs are defined in `src/defi_event_study/config.py`.

## Reproducibility note

The DefiLlama API returns the *current* full history on every call, so
re-fetching after the fact may yield slightly revised values (DefiLlama
occasionally backfills or corrects TVL). The CSV snapshots committed in
`raw/` are the exact data used in the paper (pulled April 2026).

- To replicate the paper exactly: keep the committed snapshots and run
  `python scripts/run_analysis.py` (cache is used by default).
- To refresh: `python scripts/run_analysis.py --no-cache`.
