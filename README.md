# Credit Sovereignty in DeFi Lending: Evidence from the KelpDAO Exploit

Replication code and data for the paper *"Credit Sovereignty in DeFi Lending: Evidence from the KelpDAO Exploit"*, a causal event study of how protocol-level vertical integration shapes DeFi lending resilience under a composability shock.

## Overview

On April 18, 2026, a $293M exploit of the KelpDAO LayerZero bridge cascaded into Aave V3 through its restaked-ETH (rsETH) E-Mode exposure, generating $177–230M in bad debt and $15.1B in net outflows — while Spark Protocol, which had rejected rsETH E-Mode and operates a protocol-level credit facility (Sky's Direct Deposit Module, D3M), received ~$1.3B in net inflows within seven days.

We exploit this differential exposure as a natural experiment and estimate the causal effect of protocol architecture on liquidity resilience using a four-specification **Difference-in-Differences** framework on a 131-day daily TVL panel (Aave V3, Spark, Morpho; N = 393 protocol-day observations).

### Key findings

| Result | Estimate |
|---|---|
| TVL penalty, Aave vs. Spark (preferred Spec D, lagged DV, HAC SE) | **−9.7%** (p < 0.001) |
| Upper-bound level estimate (Spec B, protocol-specific trends) | −52.9% |
| Pre-event APY variance ratio, USDC pair (Levene test) | **F = 35.1** (p < 0.001) |
| Share of TVL effect mediated by the interest-rate channel (Baron–Kenny) | **90.7%** |

The pre-event Levene tests show Spark's rate stability is a *structural* property predating the crisis by over a year; the mediation analysis shows the differential TVL response is transmitted almost entirely through the interest-rate channel. We attribute this resilience to **credit sovereignty** — the capacity to create credit endogenously (via the D3M, backed by real-world-asset collateral) rather than relying exclusively on external depositors.

## Data

All primary data are from **DefiLlama** (industry-standard on-chain metrics), accessed via its public REST API:

- **TVL** (`https://api.llama.fi/protocol/{slug}`): daily USD TVL for `aave-v3`, `spark`, `morpho`; panel window Dec 17, 2025 – Apr 27, 2026 (120 pre / 9 post days).
- **APY** (`https://yields.llama.fi/chart/{pool_id}`): matched stablecoin pools — Aave V3 USDC, Aave V3 DAI, Spark USDS, Spark Savings USDC (pool IDs in `src/defi_event_study/config.py`).
- **Event timestamp**: 2026-04-18 00:00 UTC (first exploit transaction, per Lookonchain / PeckShield).
- Supplementary: Sky governance portal and Steakhouse Financial Q1 2026 report (RWA collateral composition).

Because live API responses evolve, every fetch is cached to `data/raw/` as a CSV. The committed snapshots are the exact data used in the paper; see [`data/README.md`](data/README.md).

## Methodology

1. **DiD (Table 4).** Four specifications on log TVL: baseline with common trend (A), protocol-specific trends (B), first differences (C), and lagged dependent variable (D, preferred). All SEs are Newey–West HAC (5 lags). Specs A–B exhibit low Durbin–Watson statistics and are reported as descriptive upper bounds; Spec D (DW = 1.74) anchors the causal claim.
2. **Placebo & rolling scans (Fig. 3).** Pseudo-event dates in Jan–Mar 2026 and a rolling 35-day DiD coefficient scan confirm the effect is unique to April 18.
3. **Levene variance tests (Table 6).** Asset-paired pre-event APY variance tests establish that Spark's rate stability predates the shock.
4. **Mechanism (Table 7).** A Baron–Kenny attenuation analysis: adding APY stress to the preferred DiD reduces the event coefficient by 90.7%, consistent with near-complete mediation via the interest-rate channel. (Caveat: attenuation partly reflects mechanical correlation between APY stress and the post-event indicator; discussed in the paper.)
5. **Robustness.** Alternative control (Morpho, partial SUTVA violation noted), 30/60/90/120-day pre-windows, weekly aggregation, and a whale-adjusted Spark series removing a $174M single-address deposit at T+2.

## Limitations

- Post-event window limited to 9 days; synthetic DiD left for future work.
- Morpho control shows a partial SUTVA violation (−12.2% TVL), making Spark the cleaner counterfactual.
- APY horizon mismatch across pools and reliance on self-reported RWA data.
